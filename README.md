# Daily Code Accountability Bot

A WhatsApp-based accountability bot that forces daily programming check-ins,
tracks consistency, and captures daily learning reflections.

## Problem
I struggle with consistent daily programming practice and honest self-accountability.

## Solution
A WhatsApp bot that:
- Asks daily if I coded
- Forces reflection
- Tracks streaks
- Marks missed days clearly

## MVP Features
- Daily WhatsApp reminder
- YES/NO coding check
- Reflection questions
- Streak tracking
- Data persistence

## Tech Stack
- Node.js
- Express
- WhatsApp Cloud API
- MongoDB
